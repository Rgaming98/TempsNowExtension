# How to Install TempsNow Helper

To enable advanced thermal monitoring (CPU, GPU, RAM, SSD temperatures), you need to install the Helper Extension.

## Installation Steps

1.  **Double-click** the `TempsNowHelper.pkg` file in this folder.
2.  Follow the installer prompts (Continue -> Install).
3.  You may be asked for your password or Touch ID to allow the installation. This is required to install the secure helper tool.
4.  Once installed, open the **TempsNow** app.
5.  Click the **"?"** icon in the menu bar app to verify the extension is "Detected".

That's it! You now have full thermal monitoring enabled.
